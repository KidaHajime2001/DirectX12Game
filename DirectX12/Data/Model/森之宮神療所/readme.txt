森之宮神療所はくちばしPのオリジナルキャラクター、森之宮先生のすむ場所です。

私がモデリングした3DモデルはゆっかままんがくちばしPのミク曲「本日営業中」
の動画からインスピレーションを受けて想像して描いた建物です。
この建物の著作権はゆっかままんにありますので絶対に販売しないで下さいね。
